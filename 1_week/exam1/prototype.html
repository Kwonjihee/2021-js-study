Prototype

Array.join([separator])

  • - 배열의 모든 요소를 연결해 하나의 문자열로 만든다.
  • - separator : 배열의 각 요소를 구분할 문자열을 지정한다. 
var arr = [A, B, C];
var arr2 = [];

arr.join()

result :

- separator를 생략하면 배열의 요소들이 쉼표로 구분된다.

arr.join(' ')

result :

- separator가 빈 문자열이라면 모든 요소들이 사이에 아무 문자도 없이 연결된다.
